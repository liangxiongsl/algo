### 状态压缩dp ###

!!! 理论
	- 棋盘方案计数常见的限制：（默认对列进行压缩）
		- 同一列内的两个 1 的哈密顿距离不小于 h（但可以等于 0）
			- 可以对合法列进行 预处理 或 离散化（离散化实质就是 逆变量代换），顺便预处理 pop_count
			```title="如：哈密顿距离不小于 2"
			int id, r[1<<N], pc[1<<N];
			void dfs(int x, int j, int k){
				if(x>=1<<n) r[++id]=j, pc[id]=k;
				else dfs(x<<1, j, k), dfs(x<<2, j|x, k+1);
			}
			// dfs(1, 0, 0)
			```
		- 相邻列的哈密顿距离不小于 d
