#### bk ####

给定长度为 n 的 01 字符串 s（只包含 0 或 1），构造一个 $n\times n$ 的矩阵 S，矩阵的第 i 行为字符串 s 循环右移 $i-1$ 次后的字符串

求出满足每个元素都为 1 的最大子矩阵的面积

注：假设字符串为 $01011$，那么它可以构造出的矩阵为 $\begin{bmatrix}01011\\10101\\11010\\01101\\10110\end{bmatrix}$，可以发现满足条件的最大子矩阵的面积是 2


#### inp ####

长度为 n 的 01 字符串（$n\le 2 \cdots 10^5$）

#### out ####

一个整数，表示最大子矩阵的面积

#### test ####


```input1
01011
```

```output1
2
```

```input2
0010001110
```

```output2
4
```

```input3
010010010
```

```output3
1
```
